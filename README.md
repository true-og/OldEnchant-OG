# OldEnchant-OG

A fork of [zOldEnchant](https://github.com/Maxlego08/zOldEnchant), which allows servers to have the enchantments of the 1.7.10 in 1.19.4. It also mitigates the "too expensive" error in anvils.

![Gif of enchantment table](https://img.groupez.dev/zoldenchant/enchant.gif)

## Changes over zOldEnchant

- Updated for 1.19.4.
- Switched from maven to gradle.
